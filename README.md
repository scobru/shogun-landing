# Shogun Landing

This is the landing page for the Shogun project and its ecosystem of decentralized applications.

## Binnu

**Binnu** is the Shogun ecosystem’s decentralized P2P pastebin:

- **Decentralized storage** on the Gun.js P2P network (no central server)
- **Optional end‑to‑end encryption** via SEA for sensitive pastes
- **Code‑friendly**: syntax highlighting and dual editor modes (Monaco + simple editor)
- **Hash‑based URLs** for immutable, verifiable pastes

You can find Binnu here:

- **App**: `https://binnu.shogun-eco.xyz/`
- **Repository**: `https://github.com/scobru/shogun-binnu`
